Introduction to Next.js apps on ZEIT Now

A day-long exploration of Next.js, ZEIT Now, Slack and Pusher

Workshop by ZEIT

During the workshop we will introduce the participants to Next.js by going through the steps to build a website that handles contact form submissions and sends them to you in a Slack channel. We will also explain how external services like Slack or Pusher can be leveraged using Now and Next.js, and walk you through the process to deploy these applications on the ZEIT platform Now.

Next.js

An opinionated, feature-rich lightweight framework for static and server‑rendered applications.
“React Applications Made Simple”.

Now.sh

ZEIT Now is a cloud platform for static sites and serverless functions. It enables developers to host websites and web services that deploy instantly, scale automatically, and requires no supervision, all with no configuration.

Slack

Slack apps extend and expand the capabilities of your Slack workspace. Cultivate conversation, customize functionality, or integrate ingenious services. Slack apps enhance internal communication and automate all kinds of marvelous or mundane tasks.

Pusher

Pusher is a set of hosted APIs that makes it super-easy to quickly add real-time, secure, bi-directional functionality to web and mobile apps, or any other Internet connected device.
